# Don't Repeat Yourself - Component Library

## Code/Design review - Component Library

### Opdracht: Code & Design Review voorbereiden

Om een indruk te krijgen van hoe jullie RAPPE componenten maken, gaan wij aan de hand van een aantal FDND Agency conventies naar jullie code kijken. Om dit gericht te kunnen doen bereiden jullie de code & design review goed voor. 

### Aanpak

#### Issues aanmaken
Maak voor onderstaande FDND Agency conventies issue aan in het project board, in de kolom **Ready for review** en met het label **cdr-component-lib**. Omschrijf in ieder issue in hoeverre jullie componenten voldoen aan de betreffende conventie.
